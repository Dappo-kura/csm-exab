# リリースノート (ver 1.0.5)

今回のアップデートでは、クローズドテストの参加者様からいただいたフィードバックに基づき、ユーザー体験の向上と不具合の修正を行いました。

## 🆕 修正・改善内容

### 1. ユーザー体験の向上
- **起動時広告の撤廃**: アプリ起動直後に表示されていた広告を削除し、すぐに学習を開始できるように改善しました。
- **UIレイアウトの最適化**:
    - **下部ボタンの調整**: 端末のナビゲーションバーとボタン（通常試験、次へ、もう一度挑戦するなど）が重なる問題を修正し、誤操作を防止しました。
    - **上部「戻る」ボタンの調整**: ステータスバー（時刻・電池残量表示）とボタンが重なって反応しづらかった問題を修正しました。
    - **セーフエリア対応**: ノッチ付き端末など、様々な画面形状において最適な表示となるようレイアウトエンジンの設定を更新しました。

### 2. 機能修正
- **課金情報の取得改善**: 商品情報が取得できない場合のエラーハンドリングを強化し、より詳細なメッセージを表示するように改善しました。
